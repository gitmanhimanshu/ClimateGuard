# 🐳 Docker Quick Start

## One-Click Start (Windows)

```bash
docker-start.bat
```

## One-Click Start (Mac/Linux)

```bash
chmod +x docker-start.sh
./docker-start.sh
```

---

## Manual Start (3 Commands)

```bash
# 1. Build
docker-compose build

# 2. Run
docker-compose up -d

# 3. Open
# http://localhost:7860
```

---

## Stop

```bash
docker-compose down
```

---

## View Logs

```bash
docker-compose logs -f
```

---

## Restart

```bash
docker-compose restart
```

---

## That's it! 🚀

Dashboard: **http://localhost:7860**
